# THE UX OF LIFE

> "You are a wallet with a pulse. Let's change that."

## Overview

**The UX of Life** is a conceptual lifestyle optimization platform designed for high-net-worth individuals who feel spiritually depleted despite their material success. It frames life changes through software engineering metaphors (refactoring, debugging, latency, kernel panic).

## Project Structure

- **Manifesto**: The philosophy behind the movement.
- **Protocols**: The services offered (Diagnostic, Refactor).
- **Initiate**: The application form.

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript (Canvas animations)

## Setup

Just open `index.html` in your browser.
